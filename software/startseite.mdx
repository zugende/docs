## Karte

Auf der Karte sehen Sie aus der Vogelperspektive Ihren gesamten Strand inklusive aller aufgestellten Mietobjekte. Den Zoomfaktor und den Startpunkt können wir für Sie auf Wunsch noch einmal anpassen. 

Die Ansicht startet immer mit dem heutigen Tag. Sofern Sie sich also vor der Saison befinden, erscheint ein Hinweistext, dass derzeit keine Körbe am Strand stehen. Springen Sie daher zum frühstmöglichen Startzeitpunkt, um Ihre Aufstellung anzusehen, indem Sie auf den vorgeschlagenen unterstrichenen und gelb markierten Zeitpunkt Ihrer frühsten Aufstellung klicken. 


## Datumsauswahl

Wenn ein Gast einen längeren Zeitraum bei Ihnen buchen möchte, dann können Sie hier die Datumsauswahl öffnen und einen Zeitraum angeben. In der Ansicht werden alle aufgestellten Körbe nun nach buchbaren und nicht buchbaren Körben differenziert.

<Info>
  Sollte ein Mietobjekt nicht frei sein, so wird ein rotes <Icon icon="circle-xmark" color="red" /> über das Mietobjekt gelegt und die Farbe des Kreises ergraut. 
</Info>

### Tag-Schnellauswahl

Gäste, die vor Ort bei Ihnen buchen, haben oftmals auch den Wunsch, lediglich einen Tageskorb für heute oder morgen zu buchen. Hierfür können Sie einfach die Schnellauswahl benutzen und auf die Schaltfläche für heute bzw. morgen klicken. Klicken Sie danach den Wunschkorb an und buchen Sie den Gast ein.

### Buttons

Ganz unten auf der Karte befinden sich 4 Buttons. Folgende Funktionen stellen diese dar:

| Icon                                           | Name         | Funktion |
| ---------------------------------------------- | ------------ | -------- |
| <Icon icon="car-side" color="black" />         | Anreiseliste |          |
| <Icon color="black" icon="circle-xmark" />     | Abreiseliste |          |
| <Icon color="black" icon="filter" />           | Filter       |          |
| <Icon color="black" icon="magnifying-glass" /> | Suche        |          |

### Auslastung
Auf der rechten Seite neben den 4 Buttons finden Sie zudem eine grüne Schaltfläche, die Ihnen zunächst die prozentuale Auslastung mitteilt. Wenn Sie die Schaltfläche auswählen, erhalten Sie zu Ihrer Auslastung noch detaillierte Informationen. Zum einen absolute Zahlen sowie freie Körbe mit elektronischem Schloss und die Splittung nach Ihren angelegten Preisklassen.
