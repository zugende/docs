## Karte

Auf der Karte sehen Sie aus der Vogelperspektive Ihren gesamten Strand inklusive aller aufgestellten Mietobjekte.

## Datumsauswahl

Wenn ein Gast einen längeren Zeitraum bei Ihnen buchen möchte, dann können Sie hier die Datumsauswahl öffnen und einen Zeitraum angeben. Hiernach werden Ihnen nur freie Mietobjekte angezeigt.

<Info>
  Sollte ein Mietobjekt nicht frei sein, so wird ein rotes <Icon icon="circle-xmark" color="red" /> über das Mietobjekt gelegt.
</Info>

### Tag-Schnellauswahl

In 90% der Fällen wird ein Gast einen Wunsch für heute bzw. morgen haben. Hierfür können Sie einfach die Schnellauswahl benutzen und auf heute bzw. morgen klicken.

### Listen und Ausbuchungsanzeige

Ganz unten auf der Leiste befinden sich 4 Buttons und eine Anzeige (welche ebenfalls angeklickt werden kann). Folgend Funktionen stellen diese da:

| <Icon color="black" icon="filter" />           | Filter       |          |
| <Icon color="black" icon="magnifying-glass" /> | Suche        |          |
